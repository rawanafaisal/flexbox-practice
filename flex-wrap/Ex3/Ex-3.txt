Exercise 3 — Navbar Layout

Create a simple navigation bar that contains:

A logo (text is fine)
3 navigation links (e.g. Home, About, Contact)
A login button
Requirements:
All elements should be in one horizontal row
The logo should be aligned to the left
The navigation links should be in the center
The button should be aligned to the right
All items should be vertically aligned (same level)
Add proper spacing between elements
Add padding to the navbar to make it visually comfortable
Keep the design clean and simple
Focus:

Use Flexbox to control:

layout (left / center / right)
vertical alignment
spacing
