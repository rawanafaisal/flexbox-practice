Exercise 4 — Simple Flexbox Gallery

Create a simple gallery that contains 10 images or placeholder boxes.

Requirements:
Display all items in a row using Flexbox
When the screen becomes smaller, the items should wrap automatically onto the next line
Each item should have a fixed width of 100px
The rows should look organized and clean (no messy spacing)
Add spacing between items so they don’t stick together
Try this (important):

Test different values of:

justify-content: flex-start
justify-content: center
justify-content: space-between

👉 Observe how the layout changes each time

Focus:

Use Flexbox to practice:

wrapping (flex-wrap)
spacing (gap)
alignment (justify-content)
clean visual layout
