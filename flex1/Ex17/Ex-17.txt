Exercise 17: Box Sizing Behavior

Create 3 boxes in a row.

🎯 Requirements:
Each box has:
width: 200px;
Add:
padding
border
Observe how the size changes
🔁 Then:

Add:

box-sizing: border-box;
Observe the difference again
🧠 Goal:

Understand:

How padding و border affects on size
and how box-sizing Changes the way the size is calculated
💡 Hint (simple 👀):
.box{
	width: 200px;
	padding: 20px;
	border: 5px solid black;
}

then:

box-sizing: border-box

___________________________________________________________________________________
Answer: 
لاحظت فرق واضح للعين بس ماقدرت افهم فايدته يعني لما كان في padding كان التوسع بيروح بالعرض كل ما ازيد اما لما حطيت box-sizing: border-box العرض ما بيتاثر او شي وبيزيد padding بدةن ما يتاثر حجم البوكس او شي ... هذه ملاحظتي النظرية لكن ما قدرت استوعب الفايدة والهدف 
يعني فهمي ايش يصير هو نفسه ليش مهم اللي انا اجبت عليه انو عشان حجم البوكس الويدث ما يتاثر طيب لو كان في هايت هل box-sizing ياثر على الهايت كمان ولا واحد فيهم بس ولا بس الويدث ؟
