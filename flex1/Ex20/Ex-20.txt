Exercise 20: Spacing in a Section

Create a section that contains:

A title
A paragraph
🎯 Requirements:
Give the section space from the rest of the page using margin
Add proper spacing inside between the elements
Avoid making everything look cramped
💡 Hint (خفيف 👀):
.section{
	margin: 40px;
	padding: 20px;
}

.section p{
	margin-top: 10px;
}
🧠 Goal:

Understand:

How to combine margin (outside) and spacing between elements to create a clean layout
